# Haichao Wang Personal Website

This repository contains the Astro version of my GitHub Pages personal website, based on the Astro Nano template.

## Sections

- Home: profile, research focus, education, awards, and contact links.
- Blog: Markdown-based research notes and technical articles.
- Research: research experience timeline.
- Publications: selected publications with paper links.

## Development

```bash
pnpm install
pnpm dev
pnpm build
```

The site is configured for GitHub Pages at `https://hychaowang.github.io`.
